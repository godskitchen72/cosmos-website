# Cosmos Medical Technologies Website

Production-ready Next.js website for Cosmos Medical Technologies.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Upload this folder to GitHub.
2. In Vercel, click **Add New Project**.
3. Import the GitHub repository.
4. Framework preset: **Next.js**.
5. Build command: `npm run build`.
6. Output: handled automatically by Vercel.
7. Click **Deploy**.

## Included

- Next.js app router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- Cosmos logo assets
- Responsive enterprise SaaS landing page
